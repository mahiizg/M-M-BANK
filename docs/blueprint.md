# **App Name**: M&M Bank

## Core Features:

- User Authentication: Secure user and admin login with distinct credentials.
- Dashboard Summary: Display account balances, recent transactions, and quick-action buttons.
- Fund Transfers: Enable users to transfer funds within the bank and to other banks.
- Transaction History: Show a detailed, filterable history of all transactions.
- Profile Management: Allow users to view and modify their personal details and change passwords.
- Customer Management (Admin): Enable admin to search, view, edit, and manage customer accounts.
- Report Generation (Admin): Allow admin to generate a monthly transaction report.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) to convey trust and stability.
- Background color: Very light blue (#E8EAF6) for a clean and calm interface.
- Accent color: Golden yellow (#FFC107) for highlighting important actions and information.
- Body and headline font: 'PT Sans', a humanist sans-serif, for a balance of modern style and warmth.
- Use minimalistic line icons for navigation and actions, ensuring clarity and visual appeal.
- Employ a card-based layout with soft shadows and ample white space for a clean and organized presentation of information.
- Use subtle transitions and animations for loading states and confirmations to provide a smooth user experience.
